# C++ Banking System (File-Based + Hashing)

## Project Overview
A simple console-based banking system built with C++. It supports user account creation, login authentication, and basic banking operations like deposit, withdrawal, and balance checking. Data is stored in a local file-based database using simple hashing (DJB2 algorithm) for passwords and PINs.

## Features
- Create new account
- Secure login system
- Transaction PIN verification
- Deposit and withdraw money
- View account balance
- File-based data storage
- Basic hashing for security

## Database Format
users.txt stores data in this format:

username|hashed_password|hashed_pin|balance

## Note
This is a learning project and not intended for real-world banking use.
